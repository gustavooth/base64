# base64

ok....it's working

## Usage

```
Model: -encode: Encode input to base64
Model: -decode: Decode input to base64
Option: --verbose: Show bits

./base64 <model> <options> <str>
```

## Build

 - Run command `make`.

### Requirements for Linux

Install these via package manager:

 - git
 - make
 - clang
 - gdb // for debugging
